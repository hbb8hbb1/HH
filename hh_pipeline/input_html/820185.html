2. Stats: 不是难题，但是解题时面试官会追问的很细，所以其实这一部分个人答的没有很好

  • 已知一个正态分布 N(mu, 1)， want to test the null hypothesis that mu = 0. 如果现在只有一个样本点，如何去做这个test
       因为已经知道分布了，直接根据数据点算CI （假设alpha=0.05)， 看0是不是在CI里面

  • 接上问，如果只关注于 X >1 的样本点，基于上一个test的方法，what's the type-I in this case
       这道题当时脑子宕机了没想明白，也没有立马理解面试官想要什么，浪费了挺多时间，被hint了很多。其实面试官想问的是一个条件概率 P(X>=1.96 | X>1)
-baidu 1point3acres
  • 1000个学生记录baseball batting over a week，选 top 100 和 bottom 100 的学生，give extra training, then retest them in the following week. 问你对这个实验的看法
       回答说样本选取有bias, 没有random sample没有办法知道是学生本身的条件还是training起的作用。 追问如果这个题如果只想看training对于top学生的影响，这个实验是否合理。
