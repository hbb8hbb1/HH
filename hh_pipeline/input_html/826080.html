申请的2022年data scientist new grad，timeline如下：
9月初找人内推了positio
9/27 hr reach out联系电面，中间因为一些事情耽搁了一段时间所以最后约到了10/29店面
10/29 电面（google meet interview）
11/5 通知过了联系prep call+约final
11/29 final interview （由于时间问题，分别安排到了两天）
签了NDA，大部分问题都是open-ended的，所以就大概说说问题以及一些问题我的回答。
以下内容需要积分高于 150 您已经可以浏览

Round 1. stat+data intuition
a. 如何构建ab testing，如何收集数据，里面隐藏了一个simpson‘s paradox，如何解决。（我的回答是可以用mixed effects，然后要么分析anova，要么用regression看系数是否significance，感觉面试官挺满意这个答案)；
b. 一个跟时间相关的testing问题，一个feature launch前后的改变如何measure etc。

Round 2. data intuition + stat
聊background，research，etc，
a. 跟imbalance data相关的classification问题，如何解决（oversampling，downsampling，data augmentation，etc），如何evaluate（precision-recall curve，mis-classification rate，etc）。
b. logistic regression vs ols，系数怎么求，如何加penalty等。

Round 3. data intuition + stat
a. 面经1000个人，100个电影，评分问题。
b. sampling bias问题，如何解决。
后面还有10分钟左右，面试官说咱们再做一题，就问了下面这个题目。
c. 给你一串数，问如何检测这串数字是不是从某个分布randomly generated出来的。（我当时的思路是先检测一下有没有周期性pattern，然后看trend，连续的单调序列的长度作为test statistics之类的）

Round 4. coding
a. data manipulation
b. bootstrap confidence interval



Round 5. BQ
常规bq题目。
