1. SQL
schema: timestamp | patient_id | visit_ide
1.1 List all fields for all patients' last visit - window function
1.2 rolling count the number of active patients in the past 7 days. 意思就是数当前日期往前七天内active patient有多少个。例如日期1/7数的就是1/1-1/7的病人数，1/8数的是1/2-1/8的病人数。
我问了两个问题
a) are you counting the number of unique patients in the past 7 days or just the number of patients? -- the number of unique patients
b) Is there any gaps between visit days? like, no visits on 1/2 but we have patients on both 1/1 and 1/3. -- yes, there are gaps.

最后用的是self join。又接着说，你要最后的结果是需要每一天都有吗？他说是的。我说能在别的地方实现吗还是必须sql？他说如果不是非得在sql你准备怎么做。我说pandas的date_range然后再join
又说，如果你在database里有这个连续的日期呢？
