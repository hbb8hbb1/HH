1.        介绍自己一个统计相关的项目，问的特别细致，务必做好准备，知道项目的每个细节，不要给自己挖坑哈。主要问题是如何设计模型，避免减少confounding factors的影响，如何account for 相关的features, 如何做test,并解释结果。
2.        Given an Unfair coin, 如何生成fair probability (50%). 假设这个coin生成head概率为p, 可以考虑生成（tail, head）or (head, tail) 因为他们的概率 都是p（1-p）,如果得到(tail, tail) 或者（head,head）重新生成一个pair就好了。
3.        OLS, 把现有数据duplicate一遍，问参数estimate和方差，Confidence interval有啥变化。 根据公式参数不变，方程相当于scale了下除以sqrt(2),所以变小了。CI也变小了。
4.        OLS的assumption有哪些？linear in the predictors, 变量independent(no collinearity)， error terms’ mean为0并且similar variance(no heteroscedasticity)
5.        生成一个序列的Unif（0，1）随机数，如果当前数字小于前一个数字，则停止，否则继续生成。问这个序列的expected 长度是多少。并simulate结果。根据P(r.v<xi)=xi, xi in (0,1)
6.        解释什么是Multiple hypothesis testing, Bonferroni correction, FDR。如果有些变量是强相关的，对test会有哪些影响？more conservative.
