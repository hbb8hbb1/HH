SQL：
table A: video_id, name
table B: video_id, date, views

q1: return the video with the most views
q2: return the mode and mean of views

第一问主要就是考察window function的用法，有没有考虑到有多个videos都是最高收视率的情况。
第二问我开始理解错了，以为算average 和 mode per video，后来clarify是算整体的mean和mode。

Product Sense
q1: how to evaluate the impact of a feature
follow up: If the feature is down, how to determine if it's bc the feature is really bad or if there are somthing else going on
follow up: difference between statistic significance and practical significance; let's say two testing results with 1% metric change, one is statistic significant and the other is not, why?(这里想了半天原来是在问confidence interval)

q2:
if we plan to charge $20 per year from customers, give me your testing strategy? Start with pros and cons of this plan
follow up: how to test it, any risk
...(记不到了，都是意料之中的follow up）
