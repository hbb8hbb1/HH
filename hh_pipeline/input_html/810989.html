HR联系可以有面试DS的机会，我询问能否面RS，回答曰G家的RS只留给真正research的人，于是申请了DS。因为手里有ddl时间紧急，只一轮电面

先详细描述一个做过的data project，模型细节问得特别细，特地问了有哪些ml/dl内容，之后问问题
以下内容需要积分高于 188 您已经可以浏览

1. 有一个mean未知，variance=1的正态分布，只观察到一个样本，如何检测mean是否为0（答：正态分布cdf/标准差原则）。附加问题，如果只观察到“第一个大于1的样本“，如何检测mean是否为0（答：truncated normal cdf=cdf/truncation）
2. 有Y=a X1 + b X2的模型，X1与X2有一定关联但并不是colinear，另有Y = c(X1+X2) + d(X1-X2)模型，问：两模型系数是否相等？用有限的数据做线性回归，样本内predicted value是否相同？样本外predicted value是否相同？附加问题，如果额外的使用l2 penalty，所得系数是否相同？为什么？
3. 对学生进行测试，选取成绩最好和最不好的各100名学生参加课外训练，训练结束后再测试，来判断训练项目有没有用，这个实验设计有什么问题？能得到什么结论（答：没有对照组，针对population不明确，测试随机性和偏差，等等，这设计一大堆问题）
4. 问某个指标在10个类别里都下降了但是总体平均却上升了，为什么（答：simpson paradox，需要解释一下原理）
5. 代码：分别求1到100里奇数的平方根的和，以及偶数的平方根的和
